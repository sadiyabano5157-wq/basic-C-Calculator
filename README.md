# Basic C Calculator

This is a simple calculator program written in C.  
It performs basic arithmetic operations using user input.

## Features
- Addition
- Subtraction
- Multiplication
- Division (with zero check)
- Modulus (with zero check)

## How It Works
1. User enters two numbers
2. User selects an operation
3. Program performs calculation and shows result
